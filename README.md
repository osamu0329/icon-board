# 🌸 みんなのアイコンボード (Cute Icon Board)

Google Apps Script (GAS) と Google スプレッドシートを使用した、シンプルでかわいいアイコン投稿掲示板です。

## 🎨 特徴
- **マルチデバイス対応**: レスポンシブデザインでスマホからも快適。
- **複数画像投稿**: 一度に最大5枚までのアイコンを投稿可能。
- **自動リサイズ**: 投稿前にブラウザ側で画像を軽量化（WebP/JPEG）し、GASの制限を回避。
- **削除機能**: 投稿ごとに発行されるユニークIDにより、特定の投稿を削除可能。
- **パステルデザイン**: かわいいピンク基調のUI。

## 🛠️ セットアップ方法

### 1. Google スプレッドシートの準備
1. 新規のスプレッドシートを作成します。
2. 1行目の各列に以下のヘッダーを入力します：
   - A列: `name`
   - B列: `images`
   - C列: `date`
   - D列: `id`

### 2. Google Apps Script (GAS) の設定
1. スプレッドシートのメニューから「拡張機能」>「Apps Script」を選択します。
2. `code.gs` の内容をエディタに貼り付けます。
3. 右上の「デプロイ」ボタンから「新しいデプロイ」を選択します。
4. 種類を「ウェブアプリ」にし、以下の設定でデプロイします：
   - 次のユーザーとして実行: 「自分」
   - アクセスできるユーザー: 「全員」
5. 発行された **ウェブアプリのURL** をコピーします。

### 3. フロントエンド (index.html) の設定
1. `index.html` を開き、156行目付近の `GAS_WEB_APP_URL` を、先ほどコピーしたURLに書き換えます。
   ```javascript
   const GAS_WEB_APP_URL = "ここにコピーしたURLを貼り付け";
   ```

## 🚀 使い方
1. `index.html` をブラウザで開きます。
2. ニックネームを入力し、ファイルを選択（複数可）します。
3. 「投稿する ✨」ボタンを押すと、スプレッドシートにデータが保存され、一覧に表示されます。
4. 自分の投稿を消したい場合は「削除 🗑️」ボタンを押してください。

## 📝 注意事項
- 画像は Base64 エンコードされてスプレッドシートに直接保存されます。
- `resizeImage` 関数により、画像は幅200pxにリサイズされ、画質が調整されます。
- GASの `no-cors` モードを使用しているため、投稿直後に「送信に失敗しました」と表示される場合がありますが、実際には送信されていることが多いです。1.5秒後に自動リロードされる設定になっています。
